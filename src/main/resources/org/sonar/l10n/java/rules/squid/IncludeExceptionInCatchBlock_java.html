Always log a summary and the exception in a catch block

Noncompliant Code Example

	public void loggingWithOutException(){   
        logger.error("only the logger");
        try{
            logger.error("this is an error");
        }catch(NullPointerException e){
            logger.error("without an exception");
        }
        return;
    }

Compliant Solution

	public void loggingWithoutID(String nonsense){
        try{
            logger.error("this is an error");
        }catch(NullPointerException e){
            logger.error("with an exception provided",e);
        }
        return;
    }
