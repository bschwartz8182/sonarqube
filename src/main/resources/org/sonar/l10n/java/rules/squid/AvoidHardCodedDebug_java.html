Use a logger utility instead of System.out.println

Noncompliant Code Example

        try{
            report.generate();
        }catch(NullPointerException e){
            System.out.println("Could not generate report " + e.toString()); // Noncompliant
        }
        return;
    }

Compliant Solution

	public void loggingWithoutID(String nonsense){
        try{
            report.generate();
        }catch(NullPointerException e){
            logger.error("Could Not Generate Report",e);
        }
        return;
    }
